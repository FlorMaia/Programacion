Proyecto en GIT

  1. El primer paso:Es crear una carpeta en el escritorio después adentro de la carpeta hacemos nuestro archivo TXT a nuestro nombre.


  2. El paso dos: Después de hacer nuestro archivo txt , Abrimos el git bash y escribimos git init que significa Inicializa un nuevo repositorio Git en el directorio actual.


  3. El tercer paso:Es mover los archivos al stating área con el comando git add y lo que hace es Agregar archivos al área de preparación para la siguiente confirmación


  4. El cuarto paso:Es escribir el siguiendo comando para poder poner nuestro gmail y nombre de usuario que usamos e github. Después escribimos el comando git commit y lo que hace ese comando es crear una instantánea de los cambios rastreados y los almacena en el historial del repositorio





  5. El paso cinco:es Revisar estado del repositorio con el codigo git status y que hace ese comando es Muestra el estado actual del directorio de trabajo, los archivos preparados y los cambios sin confirmar.
